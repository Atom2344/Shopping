# shopping

HTML / CSS / JavaScriptで制作したエコバッグのEC風LPです。  
モダンなUIデザイン、LocalStorageを使ったカート機能、スクロールアニメーションを意識して制作しました。

---

## 主な機能

- モダンなランディングページ
- ハンバーガーメニュー
- カート追加機能
- LocalStorageによるデータ保存
- カートページ
- 商品削除機能
- IntersectionObserverを使ったスクロールアニメーション
- dialog要素を使ったモーダル表示

---

## 使用技術

- HTML5
- CSS3
- JavaScript（Vanilla JS）
- LocalStorage API
- IntersectionObserver API
